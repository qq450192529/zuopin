变量和参数必须以一致的顺序出现。第一个变量就是第一个被传递的参数的给定的值，以此类推。
function Myfunction(age,value){
alert('我的'+age+'是'+value);
}
onclick = "Myfunction('年龄','24')"

返回值，也就是说，将函数计算完以后的值返回给函数本身,此时函数本身带有值
function Myfunction(a,b){
c = a * b;
return c;
}
d=Myfunction(3,4);
function Myfunction3(){
alert(d);
}
onclick="Myfunction3()"

全局变量：在函数外声明的变量，直接赋值的变量(即使是在函数中)
页面关闭后变量销毁
局部变量：在函数内声明的变量
函数执行完就销毁
